# 📊 Customer Behaviour Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing customer shopping behavior using SQL, Python, and Power BI. The goal is to derive meaningful insights from customer data such as purchase trends, category performance, and demographic patterns.

---

## 🎯 Objectives
- Understand customer purchasing behavior
- Analyze revenue distribution across categories
- Identify trends based on age group, gender, and subscription
- Evaluate customer engagement using ratings

---

## 🛠️ Tools & Technologies Used
- **SQL** – Data cleaning and querying
- **Python (Pandas, Matplotlib, Seaborn)** – Data preprocessing & analysis
- **Power BI** – Dashboard creation & visualization
- **Excel/CSV** – Data source

---

## 📂 Dataset
- Customer shopping dataset (~3900 records)
- Includes:
  - Customer ID
  - Gender
  - Age Group
  - Purchase Amount
  - Category
  - Subscription
  - Shipping Type
  - Review Rating

---

## 📊 Dashboard Features
- ✅ Total Customers, Purchase Amount, Average Rating KPIs  
- ✅ Category-wise Revenue Analysis  
- ✅ Purchase Count by Category  
- ✅ Age Group Analysis (Revenue & Orders)  
- ✅ Subscription Insights  
- ✅ Interactive Filters:
  - Gender
  - Category
  - Subscription
  - Shipping Type  

---

## 🔍 Key Insights
- Clothing category generates the highest revenue
- Young age group contributes the most purchases
- Majority of customers are non-subscribers
- Average customer rating is around 3.75

---

## 📁 Project Files
- `customer_shopping_behavior.csv` → Dataset  
- `customer shopping analysis.sql` → SQL Queries  
- `Customer_Shopping_Behaviour_Analysis.ipynb` → Python Analysis  
- `Customer Shopping Dashboard.pbix` → Power BI Dashboard  

---

## 🚀 How to Use
1. Download dataset
2. Run SQL queries for analysis
